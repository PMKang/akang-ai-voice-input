# v1.3.0 — Easier first installation

Released on 2026-07-22.

- Added a custom DMG installer image with the app and an `Applications` shortcut.
- Added Chinese and English drag-to-install guidance, version, and compatibility information.
- The DMG supports both Intel and Apple silicon Macs running macOS 12 or later.
- GitHub Releases provide both DMG and ZIP: use DMG for first installation; the in-app updater continues to use the `macos` ZIP.

## Verification

- Package verification checks the signed app, universal architectures, ZIP, and DMG.

The distribution remains locally ad-hoc signed. Developer ID signing and Apple notarization are not yet complete.
