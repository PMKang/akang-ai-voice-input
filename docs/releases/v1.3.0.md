# v1.3.0

Released on 2026-07-22.

## Voice model configuration

- Added a dedicated Voice Model Configuration page with provider-separated Alibaba Cloud Model Studio and Doubao cards.
- Added selectable Alibaba Cloud realtime models: Qwen 3.5 Omni Flash Realtime, Qwen 3.5 Omni Plus Realtime, and Fun ASR Realtime.
- Added secure per-provider Keychain storage with reveal/hide controls and Alibaba connection testing.
- Fun ASR automatically synchronizes the local personal dictionary to an Alibaba Cloud hotword vocabulary. Normal setup requires only an API Key; Workspace ID is no longer shown or required.
- Qwen Omni models support writing-style prompts; Fun ASR is realtime transcription with provider-native hotwords. Doubao Streaming ASR 2.0 remains a configuration placeholder pending its dedicated WebSocket client.

## Easier first installation

- Added a custom DMG installer image with the app and an `Applications` shortcut.
- Added Chinese and English drag-to-install guidance, version, and compatibility information.
- The DMG supports both Intel and Apple silicon Macs running macOS 12 or later.
- GitHub Releases provide both DMG and ZIP: use DMG for first installation; the in-app updater continues to use the `macos` ZIP.

## Verification

- `swift test`
- `script/build_and_run.sh --build-only`
- Package verification checks the signed app, universal architectures, ZIP, and DMG.

The distribution remains locally ad-hoc signed. Developer ID signing and Apple notarization are not yet complete.
