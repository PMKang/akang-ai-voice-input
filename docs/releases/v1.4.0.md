# v1.4.0 — Voice model configuration

Released on 2026-07-22.

## Highlights

- Added a dedicated Voice Model Configuration page with provider-separated Alibaba Cloud Model Studio and Doubao cards.
- Added selectable Alibaba Cloud realtime models: Qwen 3.5 Omni Flash Realtime, Qwen 3.5 Omni Plus Realtime, and Fun ASR Realtime.
- Added secure per-provider Keychain storage with reveal/hide controls and Alibaba connection testing.
- Fun ASR automatically synchronizes the local personal dictionary to an Alibaba Cloud hotword vocabulary. Normal setup requires only an API Key; Workspace ID is no longer shown or required.
- Added active-model, configured/unconfigured, and model-capability status indicators.

## Notes

- Qwen Omni models support the app's writing-style prompt workflow.
- Fun ASR is a realtime transcription model: it uses the personal dictionary as provider-native hotwords and does not apply writing-style prompts.
- Doubao Streaming ASR 2.0 remains a visible configuration placeholder; its WebSocket recognition client is not included in this release.
