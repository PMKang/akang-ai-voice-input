# v1.2.4

## 更顺手的首次安装

- 新增自定义 DMG 安装镜像。
- 打开后直接看到“自在说”应用和 `Applications` 文件夹快捷方式。
- 提供中英文拖拽安装提示、版本号与系统兼容性说明。
- DMG 同时支持 Intel 与 Apple 芯片 Mac，最低系统版本为 macOS 12。

## 保持应用内更新兼容

- GitHub Release 同时提供 DMG 与 ZIP。
- 首次安装优先使用 DMG。
- 应用内更新继续下载名称包含 `macos` 的 ZIP，不改变现有更新逻辑。

## 安装说明

1. 下载名称包含 `macos.dmg` 的文件并打开。
2. 将 `Noboard · 自在说.app` 拖到 `Applications`。
3. 首次启动如果被 macOS 拦截，按住 `Control` 点击 App，选择“打开”并再次确认。

当前安装包仍为本地临时签名构建，尚未完成 Developer ID 签名与 Apple 公证。
