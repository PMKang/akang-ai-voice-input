# 阿康的 AI 语音输入法 v1.0.0

发布于 2026 年 7 月 12 日。

首个公开可用版本。

## 直接使用

1. 下载 `AkangVoiceInput-v1.0.0-macos.zip` 并解压。
2. 将“阿康的 AI 语音输入法.app”拖入“应用程序”。
3. 首次打开时，如 macOS 阻止启动，按住 `Control` 点击 App，选择“打开”。
4. 在设置中填入自己的阿里云百炼 API Key 和 Workspace ID，完成权限授权。

## 本版包含

- 基于 `qwen3.5-omni-flash-realtime` 的实时语音输入与整理。
- 全局快捷键、随当前屏幕显示的声波悬浮窗，以及输入框写入/剪贴板回退。
- 多套可查看和编辑的表达方式、本地历史、词典、Token 与费用估算、输入轨迹。
- 本地 Keychain 凭据保存与本地数据存储。

## 注意

- 需要 macOS 14 或更高版本，并使用自己的阿里云百炼凭据。
- `Fn` 可能与系统输入法切换冲突，推荐使用组合快捷键。
- 对于不支持 Accessibility 写入的输入框，结果会自动复制到剪贴板。
