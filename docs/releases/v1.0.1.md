# 阿康的 AI 语音输入法 v1.0.1

发布于 2026 年 7 月 15 日。

本次更新主要打磨实时反馈，并加入可选的应用更新流程。

## 更新内容

- 悬浮窗新增实时识别文字滚动展示、阿康 AI 标识与更明显的动态波形。
- 检测到环境噪声时，悬浮窗会显示“智能开启降噪”。
- “关于”页面新增版本状态、更新日志和手动检查更新入口。
- 仅在进入“关于”页面时检查一次 GitHub Release，不影响日常录音体验。
- 新版本可后台下载；用户确认重启后，应用会在可写目录中完成替换并重新打开。
- 版本号维持为 `1.0.1`，安装包以构建时间戳区分具体编译批次，方便调试与追溯。

## 发布附件

请上传以下文件至 GitHub Release：

`AkangVoiceInput-v1.0.1-0715094808-macos.zip`

应用内更新器会优先识别文件名中包含 `macos` 的 ZIP 附件。
