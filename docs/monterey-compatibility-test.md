# Noboard · 自在说：macOS 12（Monterey）兼容性测试

## 测试目的

验证 Noboard · 自在说在 macOS 12.0（Monterey）上的安装、首次配置、录音、文字写入和菜单栏入口是否可用，同时覆盖 Intel 与 Apple 芯片 Mac。

## 测试前准备

1. 确认系统版本：点击左上角  > “关于本机”，应为 macOS 12.x。
2. 记录芯片类型：在“关于本机”中记录“芯片”或“处理器”为 Apple 芯片 / Intel。
3. 准备一个可输入文字的应用，例如备忘录、微信、钉钉或浏览器文本框。
4. 请勿向任何人发送自己的 API Key、Workspace ID 或测试录音。

## 安装步骤

1. 下载测试 ZIP，双击解压。
2. 将 `Noboard · 自在说.app` 拖入“应用程序”。
3. 首次打开时，若出现“无法验证开发者”：按住 `Control` 点击 App，选择“打开”，再点击一次“打开”。
4. 在系统弹出提示时允许麦克风权限；首次写入文字时，按 App 指引允许辅助功能权限。
5. 打开“设置 > 模型与密钥”，填写自己在阿里云百炼创建的 API Key 和 Workspace ID，点击“测试连接”。

> macOS 12 不支持 App 内“开机启动”开关，这是系统限制；不影响录音和日常使用。
